# paste_dashboard.html (Local HTML Dashboard)

This is a **static HTML dashboard** generated from `paste.txt`.

## What it shows

- Summary cards (rows, columns, file stats)
- Data preview (first rows)
- Column profile (types, nulls, uniques)
- Simple text/term frequency section (useful if the file was unstructured)


Open it in a browser locally:

```bash
open paste_dashboard.html
```

Or on Linux:

```bash
xdg-open paste_dashboard.html
```
